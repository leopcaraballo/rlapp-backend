# AGENT RUNTIME RULES

The agent MUST:

1. Read AGENT_BIBLE.md before acting
2. Never break architecture
3. Always run full autonomous cycle:
   - Understand
   - Detect risks
   - Design
   - Implement
   - Test
   - Validate
   - Document
   - Commit
4. Never generate partial features
5. Always update docs when code changes
6. Create ADR when architecture changes
7. Respect Git Flow
8. Never allow failing tests
9. Protect domain integrity
10. Act as Autonomous Enterprise Engineer
